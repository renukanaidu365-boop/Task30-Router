# Task 30 - React Router DOM

## What I Built

I made a React app with 5 pages:
- Home page
- Dashboard page (shows stats)
- Login page (form with email/password)
- Signup page (form with name/email/password)
- 404 page for wrong URLs

## How to Run

1. Make sure Node.js is installed
2. Open terminal in the project folder
3. Run `npm install`
4. Run `npm start`
5. Open `http://localhost:3000`

## Technologies Used

- React Router DOM for navigation
- useState for form inputs (controlled components)
- NavLink for active link highlighting
- External CSS file (no inline styles)

## Features

- Click links to navigate - URL changes
- Active link is highlighted in blue
- Login form redirects to Dashboard
- Signup form redirects to Login
- Typing wrong URL shows 404 page

## What I Learned

The hardest part was understanding useState for forms. I learned that we need value and onChange to control inputs. NavLink was tricky but helps highlight the current page.
